# 《史记·补秦记》项目说明

## 项目背景

秦朝（前221年-前207年）作为中国历史上第一个大一统王朝，其历史记载本应丰富完整。然而，由于秦末李斯、赵高等人主导的"焚书坑儒"政策，加之秦朝短暂的国祚和汉初对秦的全面否定，导致秦史文献大量散佚。司马迁在《史记》中对秦朝的记载虽然宝贵，但仍有诸多缺失与偏颇。

本项目《史记·补秦记》旨在通过结合现代考古发现与史料研究，借助人工智能技术，尝试还原一个更为完整、客观的秦朝历史图景。

**项目当前规模**：收录各类文献 **90篇**（本纪8、世家12、列传54、表10、书6）

## 史料基础

本项目基于以下史料基础进行研究与重构：

1. **传统文献**：《史记》《汉书》《战国策》等古代文献中关于秦朝的记载

2. **考古发现**：
   - **云梦秦简**：1975年湖北云梦睡虎地出土，包含大量秦律令文书，为研究秦朝法制提供了第一手资料
   - **岳麓秦简**：湖南大学岳麓书院藏秦简，对传统"焚书"叙事提供了新的解读视角
   - **里耶秦简**：湖南里耶古城出土的秦代行政文书，展现了秦朝基层治理的真实面貌
   - **秦始皇陵及兵马俑**：为研究秦朝军事、技术水平提供了实物证据
   - **其他出土文物**：各地出土的秦代铜器、玉器、货币等

3. **现代学术研究**：国内外秦史研究的最新学术成果

## 项目特色

1. **修正传统偏见**：
   - 突破"焚书坑儒"的简单叙事，基于岳麓秦简等新材料，重新评估秦朝文化政策
   - 纠正汉代史家对秦朝的刻意丑化，如"伐湘山"等讹传

2. **填补历史空白**：
   - 基于云梦秦简等考古发现，重构秦朝法律体系的演进过程
   - 通过兵马俑等军事考古发现，还原秦军的组织结构与技术水平
   - 补充记载不详的秦朝人物传记，如胡亥生母、巴清等历史人物

3. **多维度解读**：
   - 从政治、军事、经济、文化等多角度全面审视秦朝历史
   - 关注被传统史书忽略的女性、技术官僚、异族等群体

4. **AI辅助研究**：
   - 利用人工智能技术整合分散史料
   - 通过大语言模型辅助历史场景重建与人物性格分析
   - 基于数据分析推测历史事件的可能进程

## 内容结构（共90篇）

本项目按照《史记》体例，分为本纪、世家、列传、表、书五大类：

- **本纪**（8篇）：记载秦始皇、秦二世等帝王事迹，并补充**六国末王各立一篇**（韩/赵/魏/楚/齐/**燕**）
- **世家**（12篇）：记载秦朝重要家族与六国遗族——
  - **秦系**：王氏（三代灭五国）、蒙氏（北筑长城）、樗里氏（宗室智囊）、秦公族（嬴姓宗室兴亡）、南越尉佗（赵佗和辑百越历五世93年）、公输氏（秦军工标准化）、冯氏（御史大夫谏死）
  - **六国遗族**：楚昭屈景（南迁百越）、齐田氏（海岛复国）、赵代王（代地游击）、魏信陵君（门客反秦）、孔氏（壁藏《尚书》抗秦焚）
- **列传**（54篇）：记载秦朝重要人物——
  - **政治人物**：商鞅李斯、张仪（连横破合纵）、司马错（灭蜀）、甘茂（宜阳之战）、王绾（封建郡县之辩）、赵高、吕不韦、韩非等
  - **军事将领**：尉缭（国尉/尉缭子兵法）、白起王翦、司马错、魏章（丹阳之战）、甘茂、章邯王离、李牧廉颇（赵将）、项燕（楚将）等
  - **刺客义士**：荆轲高渐离、太子丹、樊於期等
  - **技术官僚**：郑国（水利）、胡毋敬程邈（隶书）、甘德石申（天文）等
  - **隐士方士**：商山四皓、方士群体、徐巿等
  - **女性人物**：宣太后芈八子（中国第一位太后、摄政41年）、巴清（丹砂女王）、赵姬、华阳夫人、胡亥母妃、如姬、君王后等
  - **降将与叛将**：郭开、后胜、郑安平（秦首位降敌国尉）、降将批判集合（司马欣/董翳）
- **表**（10篇）：整理秦朝重要年表与谱系——
  - **时序表**：六国灭国年表、秦世系年表（非子至子婴38世）
  - **官制表**：秦相世系表（商鞅至李斯）、秦大将军世系表（白起至蒙恬）、秦国尉世系表（国尉官制与虎符军权）
  - **地理表**：郡县沿革表（36郡至48郡）、秦统一度量衡表
  - **社会表**：六国公族迁徙表、秦始皇东巡刻石表、秦陵寝封君表
- **书**（6篇）：专题研究秦朝制度与文化——
  - 书同文考（小篆推广与隶书底层实践）
  - 律法书（云梦秦简《秦律十八种》至焚书令）
  - 治国书（韩非法术势、尉缭子仁政、王绾李斯郡县之争）
  - 货殖书（半两钱统一、官营手工业、灵渠漕运）
  - 河渠书（郑国渠、都江堰、灵渠三大水利工程）
  - 兵书（尉缭子兵法、商君书战法、秦弩标准化、铬化青铜剑、军功爵制、长城）

## 项目意义

本项目不仅是简单的史料汇编，更是一次**以现代考古重构古代历史**的方法论实验。其重要性体现在以下四个层面：

### 一、填补秦史结构性空白

秦朝是中国历史上最关键的转折期——**帝制之始、郡县之创、书同文车同轨、长城灵渠驰道**——皆奠定于这短短十五年。然而秦朝国祚短暂，汉初对秦的刻意否定，加之项羽焚秦宫使"书史尽灭"（《史记·秦楚之际月表》），导致：

- **《史记》中秦的篇章严重残缺**：无《秦本纪》独立成篇（仅附于《秦始皇本纪》中），无《秦律》《秦官》等专题记述
- **六国视角叙事主导**：现存史料多从六国/汉初立场书写，秦自身的声音几乎湮没
- **三秦记等专史散佚**：自汉至唐的秦史专书尽数失传

本项目以90篇的规模，系统性地填补了这一结构性空白——**按《史记》体例补写秦史，是自班固《汉书》以来最完整的秦史数字化重建尝试**。

### 二、整合最新考古成果

近五十年的考古发现正在**彻底改写秦史认知**，但学术界至今缺乏一个将这些材料系统化、结构化的综合平台：

| 考古发现 | 对秦史的重写 | 本项目如何运用 |
|---|---|---|
| **云梦秦简**（1975） | 秦律非"严刑峻法"可以概括——实际上有精密的管理制度、工伤保障、疑案复审 | 专设《律法书》全篇解析 |
| **里耶秦简**（2002） | 秦的基层治理比想象中高效得多——"迁陵以邮行洞庭"证公文三日可达千里 | 贯穿全书，尤其《郡县沿革表》《治国书》 |
| **岳麓秦简**（2007） | "焚书"并非禁绝一切书籍，医药卜筮种树不焚；"伐湘山"乃汉人丑化 | 修正传统叙事（秦始皇本纪、书同文考） |
| **兵马俑**（1974） | 秦弩机误差≤0.2mm、青铜剑铬化防锈——秦的军工标准化超越时代 | 专设《兵书》两章 |
| **秦公大墓**（礼县/凤翔） | 证秦早期在中原体系之外有独立的青铜文化，后融入华夏 | 《秦世系年表》《秦陵寝封君表》 |
| **南越王墓**（1983） | 赵佗"和辑百越"是秦文化在岭南的成功延续 | 《南越尉佗世家》 |

学界有"简牍出土十倍于传世文献"之说。**本项目首次将这批过去四十年新出土的文献系统融入一部完整的秦史叙事中。**

### 三、方法论创新：AI辅助历史研究

本项目是**全球首个大规模尝试以AI辅助重构断代史的开源项目**，其方法论可复制到其他历史时期：

| 方法 | 说明 |
|---|---|
| **多源史料整合** | AI在数秒内交叉比对《史记》《汉书》与出土简牍、考古报告中的矛盾与印证 |
| **年代/地理/制度图谱** | 以代码化结构组织历史数据（年表→表格→人物链），便于扩展与修正 |
| **空白推理** | 基于已知史料和逻辑链，标注性推测缺失环节（如胡亥生母、焚书真相等），并**明确标记为推测而非定论** |
| **开放迭代** | GitHub开源，每次考古新发现可一键更新对应篇章——传统史书印刷后无法修订，本项目可永续更新 |

**这不是用AI取代史学家，而是用AI处理史学家无力完成的海量交叉比对工作——让人的判断力专注于最有价值的创造环节。**

### 四、历史认知的公共价值

秦史的重要性超越学术圈——**"如何评价秦始皇"是中国文化中最持久的公共辩论之一**。本项目试图提供更坚实的知识基础：

1. **去标签化**：秦不是"暴秦"二字可以概括，也不是"百代皆行秦政法"一句可以颂扬——90篇文献呈现一个具体、复杂、充满内在矛盾的秦
2. **考古说话**：秦律的精密、军工的超越时代、基层治理的高效——出土文献呈现的秦与"焚书坑儒"的秦，是同一个秦的两面
3. **兴亡样本**：秦以制度优势统一六国，又以制度僵化速亡——**这是帝制中国两千年治乱循环的原始模板**，理解秦就是理解中国

## 项目局限

尽管本项目力求客观还原历史，但仍面临以下局限：

1. **史料有限**：秦朝文献散佚严重，考古发现虽多但仍有局限
2. **解读偏差**：对出土文物的解读可能存在现代视角的投射
3. **技术限制**：AI技术虽能辅助研究，但不能完全替代历史学家的专业判断

因此，本项目内容应被视为基于现有证据的合理推测，而非确定无疑的历史事实。随着新材料的发现和研究方法的进步，内容将不断更新与修正。
