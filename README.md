## 人物画像提示词规范参考

生成提示词时请遵循 `历史人物画像规范`：

- **人物画像六要素**：体态面容、神态气质、服饰着装、道具配饰、背景环境、主色调
- **服饰要求**：交领右衽深衣/曲裾，不可用后世服饰；帝王戴冕旒，文官进贤冠，武官武弁，隐士幅巾
- **画面布局**：人物在上方居中偏左/右，姓名小字在四角之一，简介隶书在底部
- **最终提示词**：每个文件必须包含可直接复用的完整 ChatGPT Image 2 提示词
- **来源考证**：服饰冠饰须有出土实物、墓室壁画或传世古画依据
