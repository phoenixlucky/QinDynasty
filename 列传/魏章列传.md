### **魏章列传**

**魏章者，秦惠文王之将也。** 章善用兵，其功在丹阳一战。前312年，章率秦军大破楚师于丹阳，斩首八万，俘楚将屈匄，取楚汉中地六百里——**此战使秦楚实力自此逆转，楚从此一蹶不振，再无力争霸中原。**

#### **一、丹阳之战——秦楚命运的转折点**

前313年，张仪欺楚，许割商於六百里而实不予。楚怀王怒，发兵攻秦。惠文王命魏章率军迎击，樗里疾为副将，甘茂亦从征。

| **项目** | **内容** |
|---|---|
| **时间** | 秦惠文王更元十三年（前312年） |
| **秦统帅** | **魏章**（主将）、樗里疾（副将） |
| **楚统帅** | 屈匄（大将军） |
| **战场** | 丹阳（今河南丹水北岸） |
| **秦军战术** | 诱敌深入、两翼包抄 |
| **结果** | **秦军大胜，斩首八万，俘屈匄及裨将七十余人，取汉中地六百里** |

> **太史公曰**：张仪之舌诈楚，魏章之剑斩楚——一舌一剑，配合天衣无缝。**仪以欺诈使楚怒而失智，章以战阵使楚败而丧地。** 战后楚怀王益怒，悉发全国之兵复战，秦复破之于蓝田。**丹阳、蓝田两战，秦彻底摧毁了楚国作为大国的军事基础。** 章功第一！

#### **二、战绩与结局**

丹阳之战后，魏章又与樗里疾攻楚取召陵。惠文王卒，武王立，曾与章共伐楚的樗里疾继续受重用，而魏章失势。

魏章之卒年与葬所，史皆不载。**功臣战后忽隐于史——此为战国名将常见之命运。**

#### **三、历史地位**

| **对比维度** | **魏章** | **白起** |
|---|---|---|
| 时间 | 惠文王时期 | 昭襄王时期 |
| 主要对手 | 楚（丹阳之战） | 韩、魏（伊阙）、楚（鄢郢）、赵（长平） |
| 斩首 | 8万（丹阳一战） | 逾百万（约160万） |
| 官爵 | 大将（未封君） | 国尉→大良造→武安君 |
| 结局 | 不详 | 赐死杜邮 |

> **太史公案**：魏章丹阳之战，历来被掩于张仪欺楚的叙事阴影之下。**人皆知张仪以诈易地，而不知魏章以剑夺地。** 丹阳一役实为秦楚百年战争的转折点——此前楚尚有争霸之力，此后楚唯有苟延残喘。**论对秦统一之贡献，章不在白起之下，唯所遇之战名不及长平之显耳！**

#### **四、考古佐证**

1. **清华简《系年》**载："秦人败楚师于丹阳，取屈匄"——证丹阳之战与魏章之功
2. **丹阳故地**（今河南淅川一带）出土战国兵器坑，含秦式戈与楚式戟同窖，当为丹阳之战后秦军收缴战利品
