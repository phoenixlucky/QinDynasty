### **甘茂列传**

**甘茂者，下蔡人也，事秦惠文王，以通百家之说为左丞相。** 茂以宜阳之战为秦东出函谷打开了战略通道，以"息壤之盟"留下了君臣互信的千古佳话。然晚年遭谗去国，客死于魏——**以信始、以疑终，战国政治生态之残酷，尽在甘茂一身。**

#### **一、息壤之盟——互信的力量**

秦武王元年（前310），武王谓甘茂曰："寡人欲车通三川，以窥周室。若此，死不恨矣！"

茂乃请攻韩之宜阳。行前，茂恐武王中途变卦，乃引故事：

> "昔曾参处费，鲁人有与曾参同姓名者杀人。人告其母曰：'曾参杀人！'其母织自若。顷之，一人又告之，其母尚织自若。顷又一人告之，其母投杼下机，逾墙而走。**夫以曾参之贤，与其母之信也，三人疑之，其母惧焉。** 今臣之贤不若曾参，王之信臣又不如曾参之母，疑臣者非特三人，臣恐大王之投杼也！"

武王乃与甘茂盟于**息壤**（秦地）——誓不相疑。

#### **二、宜阳之战**

| **项目** | **内容** |
|---|---|
| **时间** | 秦武王三年（前308） |
| **统帅** | 甘茂 |
| **目标** | 拔韩之宜阳（今河南宜阳西） |
| **敌军** | 韩军据城死守，宜阳乃韩国西部门户 |
| **战术** | 强攻五个月，不克 |
| **危局** | **樗里疾、公孙奭在朝中谗茂，武王一度动摇** |
| **转机** | **武王忆息壤之盟，增兵卒五万，使乌获助之** |
| **结果** | 前307年拔宜阳，斩首六万，韩王入朝请和 |

> **太史公曰**：宜阳之战，甘茂之所以胜，在武王之信；武王之所以信，在息壤之盟。**战国之世，君臣猜忌如家常便饭，而武王与甘茂以息壤之誓始终不疑——此乃战国政治中最可贵的信任案例！** 宜阳既拔，秦东出之路通矣。

#### **三、去国与客死**

武王四年，武王举鼎绝膑而死。昭襄王立，甘茂以左丞相辅政。

然樗里疾、公孙奭等谗之于昭襄王，茂惧诛，去秦奔齐。路逢苏代（苏秦之弟），代为之说齐湣王，留茂于齐，以为上卿。

甘茂客死于魏，**未得归秦**。其孙甘宁，后仕吴为东吴名将——**祖孙皆以武勇智略名世，然各事其主。**

> **太史公曰**：甘茂以"曾参杀人"故事悟武王，以"息壤之盟"保信任。然昭襄王立，谗言一至，茂即不能自保。**以信取人者，亦以疑失人——此非茂之不幸，乃专制政治之必然。** 故曰：战国之士，可以立功，难全其身！

#### **四、考古佐证**

1. **宜阳故城遗址**（今河南宜阳韩城镇）：现存战国城墙，出韩式铜戈、秦式箭镞——证宜阳之战的激烈
2. **清华简《系年》**："武王元年，甘茂为左丞相，攻宜阳"——证甘茂相位和宜阳之战的纪年
3. **秦封泥"左丞相茂印"**（传世品）——甘茂为左丞相的实物
