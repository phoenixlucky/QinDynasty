# 《史记·补秦记》—— CLAUDE 项目指引

## 项目概述

本项目以《史记》体例系统重建秦朝历史，结合传统文献与现代考古发现（云梦秦简、里耶秦简、岳麓秦简、兵马俑等），共 **139篇**。

- **本纪**（17篇）：秦室三帝（秦始皇/秦二世/秦王子婴）+ 六国末王（韩/赵/魏/楚/齐/燕）+ 诸国末代（晋/鲁/宋/郑/卫/中山/越/蜀）
- **世家**（14篇）：秦系家族（王氏、蒙氏、冯氏、樗里氏、公输氏、秦公族、南越尉佗、李斯、项氏）+ 六国遗族（楚昭屈景、齐田氏、赵代王、魏信陵君、孔氏）
- **列传**（73篇）：政治/军事/技术/女性/刺客/方士/隐士/反秦群雄
- **表**（24篇）：六国灭国年表、秦世系+韩赵魏楚燕齐晋鲁宋郑卫中山越巴蜀世系年表、秦相/大将军/国尉世系、郡县沿革、公族迁徙、度量衡、陵寝封君、东巡刻石
- **书**（10篇）：书同文、律法、治国、货殖、河渠、兵书、农书、鬼谷书、黄石书、黄老书

## 写作规范

### 文件头格式

每个文献文件以 `## 标题` 开头（一级标题 `#` 保留给项目名），正文紧随其后。

### 引用考古证据

- **云梦秦简**（1975 睡虎地）→ 秦律、基层行政
- **里耶秦简**（2002）→ 邮驿、郡县治理
- **岳麓秦简**（2007）→ 焚书令修正、伐湘山讹传
- **兵马俑**（1974）→ 军工标准化、秦弩/青铜剑
- **秦公大墓**（礼县/凤翔）→ 秦早期文化
- **南越王墓**（1983）→ 赵佗和辑百越

### 标记约定

- `> 太史公曰​​：` — 司马迁评语体（注意末尾两个零宽空格 U+200B）
- `> 新证​​：` — 考古新发现引证
- `**加粗**` — 关键术语/人物名
- `*斜体*` — 文献名/书名
- 引用简牍原文加 `“中文引号”`
- `| 表格 |` — 制度对比、年表数据

### 交叉引用

跨文件引用统一使用 **Markdown 链接**，路径相对于根：
```markdown
详参 [秦始皇本纪](/本纪/秦始皇本纪.md)
参见 [兵书·秦弩标准化](/书/兵书.md)
交叉对照 [王氏世家](/世家/王氏世家.md)
```

### 语言风格

1. 文言与现代汉语 **交替使用**——叙事主干用现代汉语，人物点评/太史公语用文言
2. 每节以小标题 `#### 标题` 分段
3. 关键论断须标注史料来源（括号内标注文献名 + 篇目）
4. 推测性内容必须标注"推测"或"据考"

## 图片管理

- **新增人物画像** → 将 webp 格式图片放入 `img/` 目录，文件名与列传对应（如 `宣太后.webp`）
- **引用图片** → 在列传正文顶部插入 Markdown 图片：`![标题](../img/文件名.webp)`，另起一行加斜体注释
- **格式要求** → 统一使用 webp 格式，勿使用 png/jpg 直接引用
- **图片源文件** → 原始 png 可保留于 `img/` 供后续编辑，但文档引用必须使用 webp

## 常见任务

- **新增列传/书** → 在对应目录下新建 `.md`，更新 `/目录.md` 及相关计数器
- **拆分合传** → 新创建单人传记 `.md`，在目录中新增条目，**删除原合传文件**（不要保留空壳），更新 `/目录.md` 中的链接
- **重命名文件** → 用 `move_file` 改名，同步更新 `/目录.md` 中的链接；注意磁盘文件名与实际链接一致
- **补充考古证据** → 在相应段落插入 `> 新证​​：` 引用块，优先引用云梦秦简、里耶秦简、银雀山汉简、兵马俑、秦封泥等实物
- **扩充列传** → 优先加入考古证据、制度分析表、人物对比表；保持"太史公曰"的文言评语风格
- **更新年表** → 编辑 `/表/` 下对应文件，保持表格格式一致
- **交叉引证** → 在正文中用 Markdown 链接指向相关篇目
- **文件归类原则**：
  - 人物传记 → `列传/`
  - 家族史（三代以上）→ `世家/`
  - 帝王史 → `本纪/`
  - 制度/年表格 → `表/`
  - 专题研究 → `书/`
- **目录.md维护**：增删改文件后**必须同步更新目录链接和描述**，运行链接检查确保无死链

## 重要原则

- 不虚构史料 —— 推测部分必须明确标注
- 考古新发现优先于传统文本 —— 以出土文献修正《史记》误记时注明依据
- 多视角并存 —— 同一事件的不同解读可并列呈现
- 保持 `零宽空格` (U+200B) 标记的正确使用
