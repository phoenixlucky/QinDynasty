# 《史记·补秦记》—— AI 代理通用指引

## 项目身份

**《史记·补秦记》**——以《史记》体例、结合近五十年考古发现（云梦秦简、里耶秦简、岳麓秦简、兵马俑等），系统重建秦朝历史（前221–前207）的开源项目。共139篇文献，采用 Markdown 格式。

## 目录结构

```
/
├── 本纪/   帝王传记     17篇  (秦室三帝/六国末王/诸国末代)

├── 列传/   人物列传     73篇  (商鞅/白起王翦/反秦群雄/宣太后等)
├── 表/     年表谱系     24篇  (六国/韩赵魏楚燕齐/晋鲁宋郑卫中山越巴蜀世系)
├── 书/     专题研究      7篇  (律法/货殖/河渠/兵书/农书等)

├── 目录.md   全索引

```

## 工作约定

### 1. 文件操作

- **新建文献** → 放入对应目录（本纪/世家/列传/表/书），更新 `/目录.md`
- **修改文献** → 保持格式一致：`## 标题` 开头，每节用 `#### 标题`
- **删除文献** → 同步移除 `/目录.md` 中的对应条目
- **重命名** → 同步更新所有交叉引用链接

### 2. 图片规范

- **存放位置**：图片统一存放于 `img/` 目录
- **格式**：统一使用 **webp** 格式（`ffmpeg -i 源文件.png -qscale:v 85 目标文件.webp` 转换）
- **命名**：与文献对应，如 `宣太后.webp`、`赵姬.webp`
- **引用**：在文献正文顶部插入 `![标题](../img/文件名.webp)`，另起一行加 `*注释文字*`
- **源文件**：原始 png 可保留，但文档引用必须使用 webp

### 3. 格式规范

| 元素 | 格式 |
|---|---|
| 文献标题 | `## 标题`（二级标题） |
| 分节 | `#### 小节标题`（四级标题） |
| 太史公评语 | `> 太史公曰​​：`（注意末尾两个零宽空格） |
| 考古新证 | `> 新证​​：` |
| 重要人名/术语 | `**加粗**` |
| 书名/文献名 | `*斜体*` |
| 原文引用 | `“中文引号”` |
| 制度/数据 | `\| 表格 \|` |
| 跨文件引用 | Markdown 链接，根相对路径 |

### 3. 史料引用层次

```
第一优先：出土文献（云梦秦简 > 里耶秦简 > 岳麓秦简 > 兵马俑实测）
第二优先：司马迁《史记》
第三优先：其他传世文献（《汉书》《战国策》等）
第四优先：现代学术研究（标注出处）
推测内容：必须明确标注“推测”或“据考”
```

### 4. 交叉引用规范

跨文件引用使用根相对路径：

```markdown
详参 [秦始皇本纪](/本纪/秦始皇本纪.md)
参见 [兵书](/书/兵书.md)
对照 [王氏世家](/世家/王氏世家.md)
```

引用时注意：
- 用 Markdown 链接而非纯文本
- 保持链接路径与实际文件一致
- 重命名文件后更新所有入链

### 5. 语言与风格

- 叙事主干：现代汉语
- 评语/点评：文言风格（`太史公曰`）
- 每节简洁，避免大段无分段
- 关键论断标注史料来源（括号 + 文献名）

### 6. 零宽空格处理

文件中 `​​`（U+200B）是特意保留的格式标记，出现在：
- `> 太史公曰​​：`
- `> 新证​​：`

**不要删除或替换**这些零宽空格，它们是输出格式控制的一部分。

## 常见错误排查

| 症状 | 原因 | 修复 |
|---|---|---|
| 目录链接 404 | 文件名/路径变更未同步 | 更新 `/目录.md` 及所有入链 |
| 格式混乱 | 标题级别不一致 | 统一使用 `##` / `####` |
| 表格错位 | 列数/分隔符不匹配 | 检查 `\|` 对齐 |
| 引用无出处 | 遗漏史料标注 | 补充 `（《史记·篇目》）` 格式 |
